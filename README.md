# AI Attendance System - Backend

## 🚀 Overview
This is the backend API for the AI-Powered Face Recognition Attendance System. Built with Node.js, Express, and MongoDB.

## ✨ Features
- 🔐 JWT Authentication with Role-Based Access (Student/Teacher/Admin)
- 👤 User Registration & Login
- 🤖 Face Recognition API (Face detection, registration, recognition)
- 📊 Attendance Management (Mark, view, update, delete)
- 📚 Class Management (Create, update, delete classes)
- 👥 User Management (Admin CRUD operations)
- 📧 Email Notifications
- 🔑 Google OAuth Login

## 🛠️ Tech Stack
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB Atlas** - Cloud database
- **JWT** - Authentication
- **face-api.js** - Face recognition
- **Nodemailer** - Email notifications
- **Passport.js** - Google OAuth
